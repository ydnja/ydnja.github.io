How has Boston's Temperature Changed in the Past Few Decades?

Using NOAA daily temperature data, I display temperature data in the form of calendar heat maps, using ggplot in R,
largely thank's to TheComeOnMan's code found here: TheComeOnMan's Code
and the main code, here: Calendar Heap Map

I also plotted historical data on Boston's daily average, minumum, and maximum temperature and ran simple predictions to predict future temperature
